JavaScript 是 Web 的编程语言。
JavaScript是脚本语言
特点：不需要编译可以直接使用，由解释器来负责解释。

  8. NAN
    NaN===NaN，结果为false,这个与所有的数值都不相等，包括自己
  9. null和undefined
    null 表示值为空的意思
undefind 表示未定义
  10. 数组
    数组使用[]定义或者Arry方法
Array可以包含任意的数据类型（包括对象）
var arr=[]
new Array(1,2,3)
//方法
arr.push(); 在数组最后一位添加一个或多个元素,并返回新数组的长度,改变原数组.(添加多个元素用逗号隔开)
unshift(); 在数组第一位添加一个或多个元素，并返回新数组的长度，改变原数组。(添加多个元素用逗号隔开)
pop(); 删除数组的最后一位，并且返回删除的数据，会改变原来的数组。(该方法不接受参数,且每次只能删除最后一个)
reverse(); 将数组的数据进行反转，并且返回反转后的数组，会改变原数组
sort(); 用于对数组的元素进行排序,并返回数组。
arr.lenth ;用于获取数据长度，字符串类型与对象类型数据也可使用。

    if判断、for和while循环

  12. if判断
    let age = 3;
if (age < 3) {
    console.log('kuwa~')
} else if (age < 18) {
    console.log('haha')
} else {
    console.log('a')
}
  13. while 循环
    let age = 3;
while (age < 100) {
    age = age + 1;
    console.log(age)
}
  14. for循环
    let age = 3;
for (let i = 0; i < 100; i++) {
    age = age + 1;
    console.log(age)
}

    function函数（功能）

    function abs(x) {
    if (x >= 0) {
        return x;
    } else {
        return -x;
    }
}
//或者
let abs = function(x) {
    if (x >= 0) {
        return x;
    } else {
        return -x;
    }
}
